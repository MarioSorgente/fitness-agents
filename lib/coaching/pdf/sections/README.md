# Coaching PDF Sections

Store reusable coaching PDF section components here.

Examples include:

- Client overview and goals.
- Assessment summary.
- Weekly training schedule.
- Workout details.
- Nutrition guidance.
- Recovery and habit recommendations.
- Progress tracking.
- Safety notes, disclaimers, and appendices.

Mario should edit this folder when changing the content or layout of a specific PDF section. Global PDF theme and document layout belong in `../`.
