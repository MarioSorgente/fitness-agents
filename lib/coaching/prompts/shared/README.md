# Shared Coaching Prompt Fragments

Store prompt fragments reused by more than one coaching agent here.

Examples include:

- Brand voice and coaching tone.
- Safety and medical-disclaimer language.
- Output formatting rules.
- Client-profile summarization guidance.
- Exercise, nutrition, recovery, and behavior-change principles that should remain consistent across agents.

Mario should edit this folder when a prompt rule should apply broadly across multiple agents. Agent-only instructions belong in `../agents/`.
