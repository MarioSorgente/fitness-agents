# Coaching Form

Store all coaching intake form definitions here.

This folder is for:

- Field names, labels, descriptions, placeholders, and help text.
- Field grouping and ordering metadata for the intake experience.
- Defaults, option lists, and UI-facing validation messages.
- Form-to-schema mapping helpers that transform raw intake answers into coaching inputs.

Mario should edit this folder when adding, removing, renaming, or reorganizing intake questions. Shared data contracts should live in `../schemas/` and be imported here rather than duplicated.
